# Not Fair C++ Client

## Install

1. Download and build [socket.io-client-cpp](https://github.com/socketio/socket.io-client-cpp).
2. Set `SOCKETIO_CLIENT_CPP_BUILD_DIR` in `Makefile` to the build directory from step 1.

## Code

Write your code in `main.cpp`.

## Compile

Run `make`.

## Run

Run `./main`.
